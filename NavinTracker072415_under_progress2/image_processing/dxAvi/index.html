A Directshow based AVI interface for Matlab

 

This library contains helper functions to read image frames into Matlab from an AVI

file using DirectShow.

 

It hence,

a) does not suffer from the 2Gb file size limit of the VFW based Matlab aviread

b) handles most codecs installed on the system

c) has functions quite similar to aviread

d) but only works for windows :(

 

-- Depending on your Matlab version, the .mexw32 files may need to be renamed as .dll

 

-- testDxAvi.m is a simple example, should now pretty much run off the bat.

 

   The library usage is as follows,

 

            [avi_hdl, avi_inf] = dxAviOpen(avi_filename);

       pixmap = dxAviReadMex(avi_hdl, frame_num);

       img = reshape(pixmap,[avi_inf.Height,avi_inf.Width,3]);

       dxAviCloseMex(avi_hdl);

 

-- Use mex_cmd.m if you want to recompile the library

            BaseClasses directory is from "Direct X 9.0 SDK 2002" (Google should get you this)

            Code here is based on GrabBitmaps.cpp in DirectShow samples (there are also many many other awesome samples).

            It is quite weird and disappointing that the new DirectX SDK do not include these very useful samples.

            This library is only tested with VC++ 7 compiler (run mex -setup for this).
